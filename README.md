Calculator In Python!
This is a basic calculator with simple mathematical operations like +, -, *, /, and so on...
Every day a new mathematical operation will be added to this calculator project.

Why do this?
Started this project to kill two birds with one stone:
- To improve master in python(especially with looping)
- And to understand programming by using it to solve problems arithmetically and logically

Structure of this program:
- Operations menu
- Ensure accurate selection of operation
- Input first data type(interger)
- Input second data type(interger)
- Enter If-else loop
- Execute arithmetic task debending on selected operation
- End normally If 'Exit' operation is selected

Requirements to run:
- Terminal
- Python compatible IDE
